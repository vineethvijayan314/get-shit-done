# GSD Sketch Variant Patterns

Reusable component variants.

## Patterns
- **Card**: Hover zoom. Subtle shadow. Glass backdrop.
- **Button**: Scale down on click. Gradient background.
- **Form**: Floating labels. Smooth validation transitions.
- **Modal**: Fade in. Backdrop blur. Esc to close logic.

## Usage
1. Reference Pattern in UI Design Contract.
2. Implement via global `index.css` tokens.
