# GSD Planner Reviews

Peer review for PLAN.md.

## Objective
Audit plan quality BEFORE user approval.

## Audit Checklist
- **Atomic**: Each task < 1 hour / 10 files.
- **Verifiable**: Explicit bash/lint command per task.
- **Sequential**: Task dependencies clear.
- **Contextual**: Decisions from CONTEXT.md cited.

## Logic
- Pass: LGTM. Advance to User Approval.
- Fail: Return to `gsd-planner` for revision.
