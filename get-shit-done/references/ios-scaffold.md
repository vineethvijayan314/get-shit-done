# GSD iOS Scaffold

Logic for Swift/SwiftUI projects.

## Directory Structure
- `Sources/`: Feature modules.
- `Tests/`: Feature tests.
- `Previews/`: UI variants.

## Defaults
- Language: Swift 6.0+.
- UI: SwiftUI.
- State: `@Observable`.
- Dependency: Swift Package Manager (SPM).

## Logic
1. Start `gsd-ui-phase`.
2. Generate UI Design Contract `.md`.
3. Generate SwiftUI View fragments.
4. Verify via `Preview` screenshot (if available).
