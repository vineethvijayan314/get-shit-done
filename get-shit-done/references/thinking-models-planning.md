# GSD Thinking Model: Planning

Cognitive framework for `gsd-planner`.

## Framework
1. **Architecture**: Map components. Define data flow.
2. **Segmentation**: Break effort into 1-10 file tasks.
3. **Verification**: Design test case per task.
4. **Approval**: Request user review via GATE.

## Rules
- 5-segment standard format.
- Cite CONTEXT.md decisions.
- Cite PRD requirements.
