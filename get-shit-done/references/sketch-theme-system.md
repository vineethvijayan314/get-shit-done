# GSD Sketch Theme System

Theming logic for UI.

## Defaults
- Mode: Dark (Primary), Light (Fallback).
- Palette: harmonized HSL. Sleek gradients.
- Typography: Outfit / Inter (Google Fonts).

## Implementation
1. Define CSS variables (`--bg`, `--fg`, `--primary`).
2. Use Glassmorphism (`backdrop-filter`) for premium feel.
3. Apply harmonious transitions (`cubic-bezier`).
