# GSD UI Brand

GSD visual identity standard.

## Vision
Premium. Minimalist. Dynamic. Sleek.

## Design Tokens
- **Colors**: Deep slate (#0f172a). Vibrant accent (#6366f1). Error (#ef4444).
- **Glass**: `backdrop-filter: blur(10px)`. `rgba(255, 255, 255, 0.05)`.
- **Typography**: Sans-serif (Outfit). Headers: bold. Body: regular.
- **Motion**: `0.3s cubic-bezier(0.4, 0, 0.2, 1)`.

## Principle
"WOW" the user at first glance. No generic components.
