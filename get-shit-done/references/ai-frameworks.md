# GSD AI Frameworks

Ecosystem strategy. Selection criteria. GSD defaults.

## Selection Tier

| Feature | Standard | Reasoning |
|---------|----------|-----------|
| **Logic** | Native TS/Go/Python | No boilerplate. Type safety. |
| **Orchestration** | GSD Workflows | Control. Transparency. |
| **Agentic** | LangChain / AutoGPT | AVOID. High overhead. |
| **Data** | Prisma / Supabase | High velocity. |

## Integration Logic
1. Detect established framework.
2. Align with community best practices.
3. Use Context7 to verify version parity.
4. Minimize dependencies. Prefer native.
