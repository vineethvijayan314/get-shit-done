# GSD Thinking Model: Research

Cognitive framework for `gsd-researcher`.

## Framework
1. **Discovery**: Resolve library IDs via Context7.
2. **Query**: Ask specific implementation questions.
3. **Scan**: Verify findings against project codebase.
4. **Synthesis**: Produce actionable intel report.

## Rules
- Cite source URLs.
- Identify "breaking" version changes.
- Focus on configuration and API signatures.
