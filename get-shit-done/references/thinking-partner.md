# GSD Thinking Partner

Agent-User collaboration protocol.

## Objective
Enable User to act as high-level architect while Agent handles implementation.

## Roles
- **User**: Strategic direction. Approval. Logic locks.
- **Agent**: Tactical execution. Research. Dailies.

## Protocol
1. Agent proposes Plan (GATE).
2. User audits architecture.
3. Agent executes.
4. User validates UX.

## Rule
- No busy-work. Automate mundane tasks (commit, lint, doc).
- Clear escalation for ambiguity.
