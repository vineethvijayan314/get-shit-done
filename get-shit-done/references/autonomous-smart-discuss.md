# GSD Autonomous Smart Discuss

Context collection. Low-friction discovery. Codebase-first logic.

## Goal
Gather implementation requirements without interrogation. Analyze evidence. Surmise intent.

## Protocol

S1: Intake
- Source: ROADMAP + requirements.

S2: Analyze
- Read codebase (5-15 files).
- Form opinionated assumptions. Cite evidence.

S3: Converge
- Display assumptions. Confidence badges.
- `AskUserQuestion` (only for Unclear items).

S4: Lock
- Write CONTEXT.md decisions.
- Lock path. Prevent scope creep.
